
# Sky Jump
Compatbile with Windows, macOS and iOS  
[Download Source Code](https://github.com/Goldstriker208/Sky-Jump.git)


## Unity Editors I Used
[Unity 2017](https://unity3d.com/unity/whats-new/2017.4.40)  
[Unity 2019](https://unity3d.com/unity/whats-new/2019.4.26)  

# Controls:

Movement: WASD/Arrow Keys  

Jump: Spacebar  

Sprint: Shift
